Coreprompt

A Chrome extension that generates a structured system prompt from your current AI conversation. Continue across chat limits or switch between AI platforms without losing context.
Supported Platforms
ChatGPT, Claude, Gemini, Grok, Perplexity
How to use

1) Install the extension
2) Open any supported AI platform
3) Click the Coreprompt button
4) Copy the generated prompt
5) Paste it into any new chat

Setup
Get a free Gemini API key from aistudio.google.com
Click the extension icon and paste your API key

Built with
JavaScript, Chrome Extensions API, Gemini API
